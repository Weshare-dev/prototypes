ListJS v1.5.0

Tiny, invisible and simple, yet powerful and incredibly fast vanilla JavaScript that adds search, sort, filters and flexibility to plain HTML lists, tables, or anything.

Initialization

Table

In your class / funtion where you want to render this functionality add the following code:

componentDidMount() {
  new List(selector, {
    valueNames: ["NAME_1", "NAME_2", "NAME_3", "NAME_4", ...],
    listClass: "list"
  });
}

Where selector is your table selector and valueNames are the values that we want to be sorted from our table.

Thead

Inside the thead element you will need to make sortable the columns you specified in the valueNames by adding a data-sort attribute with the colums’s name (e.g data-sort="NAME_1") next to the .sort class.

Tbody

In order to correlate the sortable actions you will need to add the class with the same name as data-sort on the element with the value you want to sort by (e.g className="NAME_1").

For now we added the sortable functionality only on table. Soon, we will add it on other components too.
